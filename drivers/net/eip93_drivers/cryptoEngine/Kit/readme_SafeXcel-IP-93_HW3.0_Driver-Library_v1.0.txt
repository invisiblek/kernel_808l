Introduction
------------

This package contains the SafeNet EIP-93 cryptographic accelerator 
driver library(sources and documentation).

Package Content
---------------
This package contains the following components:

1. Sources
        Interfaces      - ./incl
        Implementation  - ./src/
        
2. Documentation    -   ./docs/
        Release Notes -
          SafeXcel-IP-93_HW3.0_Driver-Library_v1.0_Release-Notes_RevA.pdf
        User Guide -
          SafeXcel-IP-93_HW3.0_Driver-Library_v1.0_User-Guide_RevA.pdf
        Sample Configuration file - 
          cs_eip93.h
